Выполнен второй вариант задания:
"Написать Web-приложение с страницей «Форма обратной связи», которая
содержит поля «Фамилия», «Имя», «Отчество» (отправителя), «Получатель»
(выпадающий список из нескольких людей «ФИО» с различными email’ами), «Тема»,
«Текст сообщения». Данная форма должна отправлять с помощью AJAX всю
информацию в БД. Написать Web-страницу просмотра данных «заявок»."

Проект называется feedback.
Выполнен с помощтю таких технологий, как Spring Boot, HTML, JSP,
Jquery, AJAX, JavaScript, СУБД PosgreSQL версия 9.6 (взаимодействие с помощью
JDBC, сервер приложений Apache Tomcat 8.5.5, сборка Maven, упаковка в war.
В процессе работы над приложением использовалась система контроля версий Git.

Проект состоит из двух JSP страниц. Стартовая страница, index.jsp, содержит
форму отправки обратной связи, кнопку типа submit, <div>, в который выводится
сообщение об удачной отправке и ссылку на страницу просмотра отправленных
сообщений.
Вторая страница, viewfeedback.jsp предназначена, как упомнилась выше, для
просмотра отправленных сообщений и выводит соотвествующую таблицу.

На форме отправки ОС расположен выпадающий список для выбора адресата.
Вывод данного элемента осуществляется с помощью JSP с помощью Iterator по
коллекции экземпляров класса Recipient, где ФИО и id каждого
тега options берётся и БД получателей.

При отправке с помощью AJAX методом POST, выбор пользователя в виде id
передаётся на REST-контроллер SendController.java, ожидающий по адресу /send.

Контроллер складывает полученный ID в переменную типа int и сохраняет в БД в
таблицу feedbacks вместе с остальными параметрами.

Затем, когда пользователь переходит на страницу просмотра сообщений,
выполняется select с LEFT OUTER JOIN, который получает по данному ID
ФИО получателя и его email. Всё это складывается в экземпляр класса Feedback.java
и выводится на странице с помощью Iterator по коллекции данных экземпляров.

Как уже упоминалось, БД содержит две таблицы: FEEDBACKS и RECIPIENTS.
Первая таблица содержит данные о сообщении, создаётся и заполняется приложением
при отправке новой обратной связи.
Вторая таблица должна быть создана заранее и предзаполнена, скрипт на её
создание и заполнение лежит в папке SQL.

Файлы проекта:

index.jsp - стратовая страниа с формой обратной связи;
viewfeedback.jsp - страница для просмотра отправленных сообщений;

Application.java - стартовый класс;
WebInitializer.java - расширяет SpringBootServletInitializer, для war-упаковки;
DataBaseWorker.java - класс, содержащий методы работы с БД;
Feedback.java - класс сообщения;
Recipient.java - класс получателя;
SendController.java - REST контроллер для получения данных из формы и записи
в БД.

pom.xml - конфигурационный файл Maven.
